HTML Links

Links একটি Website এর যেকোনো পেইজ এর মধ্যে পাওয়া যায়। একটি লিংকে এর মাধ্যমে User তার নির্দিষ্ট কাজের পথ অনুসন্ধান পেয়ে থাকে।

HTML Links হচ্ছে Hyperlinks. একটি লিঙ্কে ক্লিক করে অন্য ডকুমেন্টে Movemnet করা যায়।

একটি লিংকের উপর যখন Mouse আনা হয়ে থাকে তখন Mouse এর Arrow Icon একটি ছোট হাতের Thumb Icon এ পরিণত হয়ে থাকে।

Note: একটি Link শুধুমাত্র Text হবে তা নয়। এটি যেকোনো Image File অথবা অন্য কোনো Element বহন করতে সক্ষম।

HTML Links - Syntax

HTML এর <a> ট্যাগ দ্বারা একটি Link এর পরিচিতি নির্ধারণ করা হয়ে থাকে ।

Link লিখার Syntax হচ্ছেঃ

<a href="url">link text</a>

  1. <a> ট্যাগটির সবচেয়ে গুরুত্বপূর্ণ Attributes হচ্ছে href যা কিনা একটি Link এর URL নির্ধারণ করে থাকে। href=".............." অর্থাৎ Quotation Box এর ভিতর URL এর ঠিকানা হয়ে থাকে।
  2. Link Text সেই অংশ যা পাঠকের কাছে দৃশ্যমান হবে।
  3. Link Text এ ক্লিক করা মাত্রই পাঠককে নির্দিষ্ট URL ঠিকানায় প্রেরণ করবে।

কিভাবে একটি HTML Link তৈরী করতে হয় তা নিচে উদাহরণ হিসেবে দেখানো হলোঃ

<a href="https://www.google.com/">Click Me To Visit Google!</a>

এখানে href=".............." অর্থাৎ Quotation Box এর ভিতর একটি Google Search Engine এর ঠিকানা দেয়া হয়েছে।

HTML Links - The target Attribute

Default ভাবে লিঙ্কযুক্ত Page চলমান ব্রাউজারের একই উইন্ডোতে প্রদর্শিত হবে। এটি পরিবর্তন করতে অবশ্যই লিঙ্কটির জন্য অন্য একটি লক্ষ্য নির্দিষ্ট করতে হবে।

লিঙ্কযুক্ত Page  কোথায় খুলবে তা Target Attribute নির্ধারণ করে। Target Attributes নিচের Values অনুসরন করে থাকেঃ

  1. _self  এই ভ্যালুটি একটি Link এর সাথে সংযুক্ত Documents কে Default ভাবে একই Window/Tab এর মধ্যে প্রদর্শন করে থাকে Browser.
  2. _blank এই ভ্যালুটি একটি Link এর সাথে সংযুক্ত Documents কে Default ভাবে নতুন  Window/Tab এর মধ্যে প্রদর্শন করে থাকে Browser.
  3. _parent এই ভ্যালুটি একটি Link এর সাথে সংযুক্ত Documents কে  Parent Frame এর মধ্যে প্রদর্শন করে থাকে Browser.


নিচে উদাহরণ দেয়া হলো এবং এটিতে Click করলে নতুন Tab এর মধ্যে Open হবে HTML Document .

<a href="https://www.google.com/" target="_blank">Visit Google</a>

HTML Links - Use an Image as a Link

Button as a Link

HTML Button এর মধ্যে Link ব্যবহারের ক্ষেত্রে সামান্য JavaScript কোড ব্যবহার করতে হয়। কেননা JavaScript ণিয়ন্ত্রন করে থাকে Button এ Click এর পর এটি পরবর্তীতে কি কাজ করবে।
একটি Button যদি লিংক হিসবে ব্যবহার করতে হয় তখন নিচের Syntax অনুযায়ী করতে হয়ঃ

<button onclick="document.location='' এখানে লিংক এর URL বসবে ">HTML Tutorial</button>
