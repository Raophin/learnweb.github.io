HTML Text Formatting

HTML এর মধ্যে  বিশেষ অর্থ সহ Text নির্ধারণের জন্য বেশ কয়েকটি উপাদান (Elements) রয়েছে।


নিচে উদাহরণ দেয়া হলোঃ

<p><b>This text is bold</b></p>
<p><i>This text is italic</i></p>
<p>This is<sub> subscript</sub> and <sup>superscript</sup></p>

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ

This text is bold

This text is italic

This is subscript and superscript

HTML Formatting Elements

বিন্যাসকারী(Formatting) উপাদানগুলি বিশেষ ধরণের Text প্রদর্শন করার জন্য ডিজাইন করা হয়েছিল।

  • <b> - Bold text
  • <strong> - Important text
  • <em> - Emphasized text
  • <mark> - Marked text
  • <small> - Smaller text
  • <del> - Deleted text
  • <ins> - Inserted text
  • <sub> - Subscript text
  • <sup> - Superscript text

HTML <b> and <strong> Elements

HTMl এ <b> ট্যাগটি Text Bold করার জন্যে ব্যবহৃত হয়ে থাকে।

HTMl এ <strong> ট্যাগটি Text Improtance বোঝানোর জন্যে ব্যবহৃত হয়ে থাকে। এক্ষেত্রেও Text Bold থাকে।

নিচে উদাহরণ দেয়া হলোঃ

<b>This text is bold</b>

<strong>This text is important!</strong>

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ

This text is bold
This text is important!

HTML <i> and <em> Elements


HTMl এ <i> ট্যাগটি Text Italic করার জন্যে ব্যবহৃত হয়ে থাকে।

HTMl এ <em> ট্যাগটি Text Emphasize করার জন্যে ব্যবহৃত হয়ে থাকে। অর্থাৎ User এর পড়ার সুবিধার্থে Text এর উপর জোর দেয়া হয়। এক্ষেত্রেও ভিতরে থাকা সামগ্রীটি সাধারণত Italic ভাবে প্রদর্শিত হয়।

নিচে উদাহরণ দেয়া হলোঃ

<i>This text is italic</i>
<em>This text is emphasized</em>

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ

This text is italic
This text is emphasized

HTML <small> Element

HTMl এ <small> ট্যাগটি Text ছোট  করার জন্যে ব্যবহৃত হয়ে থাকে।

নিচে উদাহরণ দেয়া হলোঃ

<small>This is some smaller text.</small>

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ

This is some normal text.

This is some smaller text.

HTML <mark> Element

HTMl এ <mark> ট্যাগটি Text Highlight করার জন্যে ব্যবহৃত হয়ে থাকে।

নিচে উদাহরণ দেয়া হলোঃ

<p>Do not forget to buy <mark>milk</mark> today.</p>

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ

Do not forget to buy milk today.

HTML <del> Element

HTMl এ <del> ট্যাগটি HTMl Doc. থেকে Text মুছে ফেলার জন্যে ব্যবহৃত হয়ে থাকে। এক্ষেত্রে Text এর মাঝামাঝি একটি দাগ টানা থাকে।

নিচে উদাহরণ দেয়া হলোঃ

<p>My favorite color is <del>blue</del> red.</p>

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ

My favorite color is blue red.

HTML <sub> Element

HTMl এ <sub> ট্যাগটি HTMl Doc. থেকে Text এর Subscript লিখার জন্যে ব্যবহৃত হয়ে থাকে। 

নিচে উদাহরণ দেয়া হলোঃ

<p>This is <sub>subscripted</sub> text.</p>

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ

This is subscripted text.

HTML <sup> Element

HTMl এ <sup> ট্যাগটি HTMl Doc. থেকে Text এর Superscript লিখার জন্যে ব্যবহৃত হয়ে থাকে। 

নিচে উদাহরণ দেয়া হলোঃ

<p>This is <sup>superscripted</sup> text.</p>

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ

This is superscripted text.
