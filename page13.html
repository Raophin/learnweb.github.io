HTML Block and Inline Elements


প্রতিটি HTML Elements এর Display মান(Values) থাকে প্রদর্শন করার জন্যে এবং এটি নির্ভর করে এটি কী ধরণের Element তার উপর।

দুটি Display Value রয়েছেঃ Block এবং Inline।

Block-level Elements

  1. একটি ব্লক-স্তর(Block Level) উপাদান সর্বদা একটি নতুন লাইনে শুরু হয়।
  2. Block Level উপদানটি সর্বদা Screen এর Full Width দৃশ্যমান হয়ে থাকে এবং বাম দিক থেকে ডান দিক পর্যন্ত প্রসস্ত থাকে।
  3. <div> Element হচ্ছে একটি Block Level এলিমেন্ট।

Block Elements Lists:

<address> <article> <aside> <nav>
<blockquote> <canvas> <dd> <main>
<div> <dl> <dt> <noscript>
<fieldset> <figcaption> <figure> <ol>
<footer> <form> <h1>-<h6> <p>
<header> <hr> <li> <section>

Inline Elements

Inline Elements মূলত Block Elements এর ভিতরে থাকে এবং এটি নতুন লাইনের মাধ্যমে শুরু হয় না । এটি প্রয়োজন মতো Width নিয়ে থাকে। 

Inline Elements Lists:

<a> <abbr> <acronym> <b>
<bdo> <big> <br> <button>
<cite> <code> <dfn> <em>
<i> <img> <input> <label>
<object> <output> <span> <textarea>
<var> <script> <script> <tt>

Ordered HTML List

HTMl এ Ordered Lists শুরু হয় <ol> ট্যাগ দিয়ে এবং প্রতিটি Lists Items শুরু হয়ে থাকে <li> ট্যাগ দিয়ে। Ordered List এর Lists Items গুলো Default ভাবে 1, 2, 3 এইভাবে নাম্বারিং এর মাধ্যমে প্রদর্শিত হয়ে থাকে।

নিচে উদাহরণ দেয়া হলোঃ

<h2>An ordered HTML list</h2>
<ol>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>  

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ

An Ordered HTML list

  1. Coffee
  2. Tea
  3. Milk

Ordered HTML List

HTMl এ Unordered Lists শুরু হয় <ul> ট্যাগ দিয়ে এবং প্রতিটি Lists Items শুরু হয়ে থাকে <li> ট্যাগ দিয়ে। Unordered List এর Lists Items গুলো Default ভাবে Circle Type বুলেট এর মতো প্রদর্শিত হয়ে থাকে।

নিচে উদাহরণ দেয়া হলোঃ

<h2>An unordered HTML list</h2>

<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>  

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ
