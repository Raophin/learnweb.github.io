HTML Basics

১) <!DOCTYPE html> এটি দ্বারা ঘোষনা করতে হয় যে এটি HTML Document এবং এটি Browsers এ সুন্দর ভাবে ওয়েব পেজ     গুলি প্রদর্শন করতে সহায়তা করে। এটি অবশ্যই HTML File এর শুরতে শুধুমাত্র একবার ব্যাবহার করতে হয় । এটি কোনো Case     Sensitive না অর্থাৎ ছোট/বড় হাতের বর্ণে লিখলেও চলে। এটির কোনো Closing Tag নেই।

২) <title>Page Title Here </title> এটি Browsers এ Websites এর Title দৃশ্যমান করে।

৩) <head> </head> এটি Browsers এ Non-Visual কার্যকলাপ করে থাকে। যেমন- এটি একটি HTML File এর সাথে অন্যান File     এর সংসযোগ স্থাপন করা, এবং HTML পৃষ্ঠা সম্পর্কে মেটা সম্পর্কিত তথ্য বহন করে।

৪) HTML Document শুরু হয় <html> এটি দিয়ে এবং  </html> শেষ হয় এটি দিয়ে।

৫) HTML Document এর দৃশ্যমান অংশ <body> (Opening Tag) এবং </body>(Closing Tag) এর মধ্যে থাকে।
   


একটি সাধারণ এইচটিএমএল ডকুমেন্ট নিচে দেখানো হলোঃ

 <!DOCTYPE html>

<html lang="en">

        <head>
                <title>Page Title</title>
        </head>
<body>
         <p>This Is Simple Paragraph</p>
</body> 

</html> 

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ

HTML Elements

HTML Elements মূলত < > এবং </> এই দুটি কোনাকুনি Brackets এর মধ্যে লিখা কিছু Characters বুঝিয়ে থাকে।
যেমনঃ <body> , <p>, <h1> এদের Elements বলা হয় ।

প্রতিটি Elements এর একটি Opening Tag ও একটি Closing Tag থাকে। এছাড়াও প্রতিটি Elements আলাদা আলাদা কাজের নির্দেশনা বহন করে থাকে। আর এই < > ...........</ > Opening Tag ও Closing Tag এর মধ্যেই লিখিত বা অন্যান তথ্য বহন করে থাকে।

HTML Attributes

HTML Attributes মূলত HTML Doc. উপাদানগুলির (Elements) সম্পর্কে অতিরিক্ত তথ্য সরবরাহ করে থাকে।
উদাহরণ স্বরুপ বলা যায় <img> ট্যাগ এর জন্যে অতিরিক্ত তথ্য দিতে পারে এমন ২ টি Attributes হলো <width> & <height> ।
Example: < img src="Myimage" width="100px" height="50px" >
অর্থাৎ এক্ষেত্রে Image File এর চওড়া হবে 100 Pixel এবং উচ্চতা হবে 50 Poxel.

  • সমস্ত HTML উপাদানগুলির(Elements) Attributes থাকতে পারে।
  • Attributes উপাদানগুলি সম্পর্কে অতিরিক্ত তথ্য সরবরাহ করে থাকে।
  • Attributes সর্বদা শুরু ট্যাগে ( < ...... > ) নির্দিষ্ট করা হয় থাকে।
  • Attributes সাধারণত Name="Value" এইভাবে লিখতে হয় Opening Tag এর ভিতরে।

HTML Headings

HTML Headings মূলত <h1> থেকে <h6> পর্যন্ত ম্মোট ৬ টি Tags বুঝিয়ে থাকে। এদের Closing Tags রয়েছে।
<h1> এটি সবচেয়ে গুরুত্বপূর্ণ এবং বড় হেডিং নির্দেশ করে থাকে ।
<h6> এটি সবচেয়ে কম গুরুত্বপূর্ণ এবং ছোট হেডিং নির্দেশ করে থাকে ।

নিচে উদাহরণ দেয়া হলোঃ

<h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>
<h4>This is heading 4</h4>
<h5>This is heading 5</h5>
<h6>This is heading 6</h6>

HTML Paragraphs

HTML সব সময় <p> ট্যাগ দ্বারা Paragraph নির্দিষ্ট করে থাকে।
একটি Paragraph সব সময় নতুন লাইনের মাধ্যমে শুরু হয়ে থাকে। এটি সাধারন Paragraph Text Blocks হিসেবে ব্যবহৃত হয়।
Web Browser স্বয়ংক্রিয়ভাবে একটি Paragraph এর আগে এবং পরে কিছু ফাকা স্থান (একটি মার্জিন) যুক্ত করে।

নিচে উদাহরণ দেয়া হলোঃ
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ

This is a paragraph.
This is another paragraph.

HTML Display


এইচটিএমএল কীভাবে Screen এ প্রদর্শিত হবে তা আপনি নিশ্চিত হতে পারবেন না। বড় বা ছোট Screen এবং Resized উইন্ডোগুলি বিভিন্ন রকম ফলাফল প্রদর্শন করে থাকে।
এইচটিএমএল কোডে অতিরিক্ত Space(ফাঁকা স্থান) যুক্ত করে দিলেও তা Browser এ প্রদর্শন হয় না। Web Page প্রদর্শিত হওয়ার পরে ব্রাউজার স্বয়ংক্রিয়ভাবে অতিরিক্ত স্থান সরিয়ে ফেলে।

নিচে উদাহরণ দেয়া হলোঃ
<p>
This             paragraph
contains a             lot of spaces
in the         source         code,
but         the             browser
ignores             it.
</p>

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ

This paragraph contains a lot of spaces in the source code, but the browser ignores it.

HTML Horizontal Rules

<hr> ট্যাগটি HTML পৃষ্ঠায় একটি থিম্যাটিক বিরতি সংজ্ঞা দেয় এবং প্রায়শই একটি অনুভূমিক একটি রেখা তৈরী করে।
<hr> উপাদানটি HTML পৃষ্ঠায় যত Contents আলাদা করতে ব্যবহৃত হয়।
<hr> ট্যাগটি একটি Blank Tag, যার অর্থ এটির কোনও </> (Closing Tag) নেই।


নিচে উদাহরণ দেয়া হলোঃ

<h1>This is heading 1</h1>
<p>This is some text.</p>
<hr>
<h2>This is heading 2</h2>
<p>This is some other text.</p>
<hr>
<h2>This is heading 2</h2>
<p>This is some other text.</p>

This is some text.

HTML Line Breaks


এইচটিএমএল <br> উপাদানটি একটি লাইন বিরতি নির্ধারণ করে।
নতুন Paragraph শুরু না করেই যদি একটি লাইন ব্রেক (একটি নতুন লাইন) দেয়ার প্রয়োজন হয় তবে <br> ব্যবহার করতে হয়।


নিচে উদাহরণ দেয়া হলোঃ


<!DOCTYPE html>
<html>
<body>

<p>This is <br> a paragraph <br> with line breaks.</p>

</body>
</html>

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ

This is
a paragraph
with line breaks.
