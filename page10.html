HTML Images

Images একটি Website এর Visual Appearance আরো সুন্দর করে তুলতে পারে। <img> এর ট্যাগের মাধ্যমে একটি Web Page এর মধ্যে Image Attach করতে হয়। আর এই ট্যাগের কোনো Closing Tags নেই। <img> এর ট্যাগের ভিতরে ২টি Attributes থাকা আবশ্যিক এবং তা হলো src এবং alt এর উপস্থিতি।

১) src এটি Image File এর Location/URL বুঝিয়ে থাকে অর্থাৎ Image ফাইলে কোন ফোল্ডার এ আছে তা চিনিয়ে দিতে হয় এখানে ।

২) alt এটি হচ্ছে Image File সম্পর্কে অতিরিক্ত তথ্য বহন করে থাকে। যেমন; কোনো কারনে Image Load না হলে তখন এই alt এর মধ্যে যা লিখা থাকবে তা    প্রদর্শন করিবে।


<img> ট্যাগ ব্যবহারের Syntax নিচে লিখা হলোঃ

<img src="Image ফাইলের URL এখানে হবে" alt="alternatetext">

Image Size - Width and Height

Use an Image as a Link
