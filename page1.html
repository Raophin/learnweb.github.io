HTML Introduction
(HTML পরিচিতি)

Web Development শিখার শুরুতেই HTML দিয়ে শেখা শুরু করতে হয়। কেননা একটি ওয়েব পেজের যা কিছু দৃশ্যমান তার মূল কাঠামো তৈরী করা হয় HTML দিয়েই । তাই আমরা HTML দিয়েই কোর্স শুরু করবো এবং ধাপে ধাপে শিখবো।

(মৌলিক ধারনা) 

1
HTML ওয়েব পৃষ্ঠাগুলি তৈরির জন্য আদর্শ মার্কআপ ভাষা।

2
HTML অর্থ হাইপার টেক্সট মার্কআপ ল্যাঙ্গুয়েজ (Hyper Text Markup Language)
3
HTML একটি ওয়েব পেজের গঠন বর্ণনা করে।
4
HTML বিভিন্ন উপাদানের সমন্বয়ে গঠিত।
5
HTML উপাদানগুলি কীভাবে প্রদর্শন করতে হয় তা ব্রাউজারকে বলে 


একটি সাধারণ এইচটিএমএল ডকুমেন্ট নিচে দেখানো হলোঃ

 <!DOCTYPE html>
<html lang="en">
<head>
<title>Page Title</title>
</head>
<body>
<p>This Is Simple Paragraph</p>
</body> 
</html>
