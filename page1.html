HTML Introduction
(HTML পরিচিতি)

Web Development শিখার শুরুতেই HTML দিয়ে শেখা শুরু করতে হয়। কেননা একটি ওয়েব পেজের যা কিছু দৃশ্যমান তার মূল কাঠামো তৈরী করা হয় HTML দিয়েই । তাই আমরা HTML দিয়েই কোর্স শুরু করবো এবং ধাপে ধাপে শিখবো।

(মৌলিক ধারনা) 

1
HTML ওয়েব পৃষ্ঠাগুলি তৈরির জন্য আদর্শ মার্কআপ ভাষা।

2
HTML অর্থ হাইপার টেক্সট মার্কআপ ল্যাঙ্গুয়েজ (Hyper Text Markup Language)
3
HTML একটি ওয়েব পেজের গঠন বর্ণনা করে।
4
HTML বিভিন্ন উপাদানের সমন্বয়ে গঠিত।
5
HTML উপাদানগুলি কীভাবে প্রদর্শন করতে হয় তা ব্রাউজারকে বলে। 


একটি সাধারণ এইচটিএমএল ডকুমেন্ট নিচে দেখানো হলোঃ

 <!DOCTYPE html>

<html lang="en">

        <head>
                <title>Page Title</title>
        </head>
<body>
         <p>This Is Simple Paragraph</p>
</body> 

</html> 

Result:

This Is Simple Paragraph

উপরের HTML Document নিচে বর্ননা করা হলোঃ

  1. <!DOCTYPE html>  এখানে বোঝায় যে এটি একটি HTML Document.
  2. <html> উপাদানটি HTML পৃষ্ঠার মূল উপাদান হিসেবে ধরা হয়।
  3. <head> উপাদানটি HTML পৃষ্ঠা সম্পর্কে মেটা সম্পর্কিত তথ্য বহন করে।
  4. <title> উপাদানটি HTML পৃষ্ঠার জন্য একটি নাম নির্দেশ করে (যা ব্রাউজারের টাইটেল বারে বা পৃষ্ঠার ট্যাবে প্রদর্শিত হয়)
  5. <body> উপাদানটি HTML পৃষ্ঠার মূল অংশ নির্দেশ করে অর্থাৎ যা কিছু প্রদর্শিত হয় ব্রাউজারে।(যেমন-Title, Paragraph, Images, Links, Table, List ইত্যাদি)
  6. <p> উপাদানটি সাধারণ অনুচ্ছেদ বা লিখা প্রদর্শন করে।

Web Browser
(ওয়েব ব্রাউজার)

ওয়েব ব্রাউজার (Chrome, Edge, Firefox, Safari) উদ্দেশ্য হলো HTML File পড়া এবং সেগুলি সঠিকভাবে প্রদর্শন করা।
একটি ব্রাউজার HTML ট্যাগগুলি প্রদর্শন করে না, তবে কীভাবে ট্যাগগুলির ভিতরকার তথ্য প্রদর্শন করতে হয় তা নির্ধারণ করে থাকে।

HTML Page Structure
(HTML Page এর গঠন)

(একটি HTML Document এর কাঠামো)

দ্রষ্টব্য: <body> ট্যাগের অভ্যন্তরের সামগ্রী একটি ব্রাউজারে প্রদর্শিত হবে। <title> উপাদানটির ভিতরে থাকা সামগ্রীটি ব্রাউজারের Title Bar বা পৃষ্ঠার ট্যাবে প্রদর্শিত হবে।

ওয়ার্ল্ড ওয়াইড ওয়েবের প্রথম দিন থেকেই HTML এর অনেকগুলি সংস্করণ রয়েছে এবং এগুলির মধ্যে HTML5 আধুনিক সংস্করন। আমাদের সমগ্র কোর্সটি HTML5 এর উপর ভিত্তি করেই লিখা হয়েছে
