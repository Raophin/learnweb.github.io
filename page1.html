HTML Introduction

HTML একটি হাইপার টেক্সট মার্কআপ ল্যাঙ্গুয়েজ যা Web Pages এবং Web Application তৈরি করতে ব্যবহৃত হয়।
Web Development শিখার শুরুতেই HTML দিয়ে শেখা শুরু করতে হয়। কেননা একটি ওয়েব পেজের যা কিছু দৃশ্যমান তার মূল কাঠামো তৈরী করা হয় HTML দিয়েই । 

1
HTML ওয়েব পৃষ্ঠাগুলি তৈরির জন্য আদর্শ মার্কআপ ভাষা।

2
HTML অর্থ হাইপার টেক্সট মার্কআপ ল্যাঙ্গুয়েজ (Hyper Text Markup Language)
3
HTML একটি ওয়েব পেজের গঠন বর্ণনা করে।
4
HTML বিভিন্ন উপাদানের সমন্বয়ে গঠিত।
5
HTML উপাদানগুলি কীভাবে প্রদর্শন করতে হয় তা ব্রাউজারকে বলে। 


একটি সাধারণ এইচটিএমএল ডকুমেন্ট নিচে দেখানো হলোঃ

 <!DOCTYPE html>

<html lang="en">

        <head>
                <title>Page Title</title>
        </head>
<body>
         <p>This Is Simple Paragraph</p>
</body> 

</html> 

Result:

This Is Simple Paragraph

উপরের HTML Document নিচে বর্ননা করা হলোঃ

  1. <!DOCTYPE html>  এখানে বোঝায় যে এটি একটি HTML Document.
  2. <html> উপাদানটি HTML পৃষ্ঠার মূল উপাদান হিসেবে ধরা হয়।
  3. <head> উপাদানটি HTML পৃষ্ঠা সম্পর্কে মেটা সম্পর্কিত তথ্য বহন করে।
  4. <title> উপাদানটি HTML পৃষ্ঠার জন্য একটি নাম নির্দেশ করে (যা ব্রাউজারের টাইটেল বারে বা পৃষ্ঠার ট্যাবে প্রদর্শিত হয়)
  5. <body> উপাদানটি HTML পৃষ্ঠার মূল অংশ নির্দেশ করে অর্থাৎ যা কিছু প্রদর্শিত হয় ব্রাউজারে।(যেমন-Title, Paragraph, Images, Links, Table, List ইত্যাদি)
  6. <p> উপাদানটি সাধারণ অনুচ্ছেদ বা লিখা প্রদর্শন করে।

Web Browser

ওয়েব ব্রাউজার (Chrome, Edge, Firefox, Safari) উদ্দেশ্য হলো HTML File পড়া এবং সেগুলি সঠিকভাবে প্রদর্শন করা। একটি ব্রাউজার HTML ট্যাগগুলি প্রদর্শন করে না, তবে কীভাবে ট্যাগগুলির ভিতরকার তথ্য প্রদর্শন করতে হয় তা নির্ধারণ করে থাকে।

(একটি ওয়েব ব্রাউজারের ছবি)

HTML Page Structure

(একটি HTML Document এর কাঠামো)

দ্রষ্টব্য: <body> ট্যাগের অভ্যন্তরের সামগ্রী একটি ব্রাউজারে প্রদর্শিত হবে। <title> উপাদানটির ভিতরে থাকা সামগ্রীটি ব্রাউজারের Title Bar বা পৃষ্ঠার ট্যাবে প্রদর্শিত হবে।

HTML Editor

Visual Studio Code ব্যবহার করে HTML শিখুন এবং Practice করুন খুব সহজেই।

১) প্রথমে Google এ Visual Studio Code লিখে সার্চ করুন।
২) তারপর নিচের চিত্রে দেখানো লিংকে প্রবেশ করুন।
৩) তারপর নিচের চিত্রে দেখানো Download এ চাপ দিন এবং Install করে নিন আপনার কম্পিউটারে।
৪) তারপর আপনার কম্পিউটারে Installed করা Editor চালু করুন।
৫) তারপর উক্ত Extension Icon এ চাপ দিয়ে Extension Bar চালু করুন।
৬) তারপর উক্ত Extension Search Bar এ চাপ দিয়ে Brackets Keymap লিখে সার্চ করুন ও Install করুন।
৭) তারপর Install হয়ে গেলে নতুন করে Software কেটে দিয়ে চালু করুন । তারপর আপনার Computer Desktop এ একটি Folder তৈরি করুন যেকোনো নাম দিয়ে (Ex: Web Practice নাম দিয়ে খুলতে পারেন) ।
৮) তারপর Visual Code এর উপরের মেনু বারের বাম পাশ থেকে File> Open Folder > আপনার তৈরী করা Folder Select করুন।
৯) Folder Select করুন।
১০) তারপর Visual Code এর বাম পাশ থেকে Explorer Icon Click করে আপনার ফোল্ডার সিলেক্ট করে Folder এর উপরে ছোট ন্যাভ বার থেকে File Icon Clikc করে এর ভিতর index.html লিখে একটি File Create করুন।

১১) তারপর Visual Code এর index.html ফাইলের ভিতর নিচের কোডটি উদাহরণ হিসেবে লিখতে পারেন এবং আপনার Desktop এ উক্ত ফোল্ডারে HTML File ডাবল Click করলে Result প্রদর্শন হবে।
আর এভাবে আপনার Editor সেট করা হয়ে যাবে পরবর্তীতে কাজ করার জন্যে ।

            <head>
                    <title>Page Title</title>
            </head>
<body>
        <p>Simple Paragraph</p>
</body>
</html> >
