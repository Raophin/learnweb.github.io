HTML RGB and RGBA Colors

RGB Color মূলত Red, Green, Blue এর ৩ কালার বুঝিয়ে থাকে। 
RGBA হচ্ছে RGB Color এর Extension হিসেবে Alpha Channel (Opacity) বুঝানোর জন্যে।
rgb(red, green, blue) এই ৩ কালারের মাধ্যমে Color Value ব্যবহার করা যায় HTML এর মধ্যে।
এক্ষেত্রে প্রতিটি Parameter( Red Or Green Or Blue ) এর Value Range হয়ে থাকে 0 থেকে 255 পর্যন্ত।
এর অর্থ হল 256 x 256 x 256 = 16777216 সম্ভাব্য Color রয়েছে!
