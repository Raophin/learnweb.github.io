Define an HTML Table

1

<table> Tag.

<table> ট্যাগ দ্বারা HTML Table নির্ধারন করতে হয়।
2

<tr> Tag.

প্রতিটি Table এর Row(সারি) নির্ধারণ করতে <tr> ট্যাগ ব্যবহার করা হয়ে থাকে।
3

<th> Tag.

প্রতিটি Table এর Header অর্থাৎ প্রথম সারির প্রতিটি কলাম এর Data কে Heading Name হিসেবে নির্ধারণ করতে <th> ট্যাগ ব্যবহার করা হয়ে থাকে। যেমন; উপরের Table উদাহরণ এর Company, Contact, Country হচ্ছে উক্ত টেবিলের Heading।

4

<td> Tag.

প্রতিটি Table এর Data/Cell নির্ধারণ করতে <td> ট্যাগ ব্যবহার করা হয়ে থাকে। 

নিচে একটি উদাহরণ দেয়া হলোঃ

<h2>Basic HTML Table</h2>
<table style="width:100%">
  <tr>
    <th>Firstname</th>
    <th>Lastname</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Jill</td>
    <td>Smith</td>
    <td>50</td>
  </tr>
  <tr>
    <td>Eve</td>
    <td>Jackson</td>
    <td>94</td>
  </tr>
  <tr>
    <td>John</td>
    <td>Doe</td>
    <td>80</td>
  </tr>
</table>
