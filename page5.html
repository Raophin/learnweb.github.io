HTML Styles

HTML Style বৈশিষ্ট্যটি মূলত কোন Elements এর, যেমন রঙ, ফন্ট, আকার এবং আরও কিছুতে স্টাইল যুক্ত করতে ব্যবহৃত হয়।


নিচে উদাহরণ দেয়া হলোঃ

<!DOCTYPE html>
<html>
<body>
<p>I am normal</p>
<p style="color:red;">I am red</p>
<p style="color:blue;">I am blue</p>
<p style="font-size:50px;">I am big</p>
</body>
</html>

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ

I am normal


I am red


I am blue


I am big

The HTML Style Attribute

এইচটিএমএল উপাদানটির Style সেট করা, Style Attribute ব্যবহার করা হয়।

এইচটিএমএল Style Attribute নিম্নলিখিত সিনট্যাক্স রয়েছে:
<tagname style="property:value;">

(ব্যাখ্যাঃ এখানে tagname হচ্ছে HTML এর যেকোন Element এর নাম হবে। property হচ্ছে CSS এর Property এবং value হচ্ছে CSS ভ্যালু।)

Background Color

CSS এর background-color বৈশিষ্ট্যটি এইচটিএমএল উপাদানগুলির জন্য Background Color নির্ধারণ করে।

নিচে উদাহরণ দেয়া হলো যেখানে একটি পৃষ্ঠার Background Color হবে #fee885 :

<!DOCTYPE html>
<html>
<body style="background-color:#fee885;">
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
</body>
</html>

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ

This is a heading

This is a paragraph.


দুটি পৃথক উপাদানের জন্য Background Color সেট করে দেখানো হলোঃ

<h1 style="background-color:powderblue;">This is a heading</h1>
<p style="background-color:tomato;">This is a paragraph.</p>

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ

This is a heading


This is a paragraph.

নিচে উদাহরণ দেয়া হলোঃ
<h1 style="color:blue;">This is a heading</h1>
<p style="color:red;">This is a paragraph.</p>

Fonts

CSS এর font-family এই Property একটি HTML উপাদানগুলির টেক্সট লিখা কিরূপ হবে তা Font-Family নির্ধারন করে।

নিচে উদাহরণ দেয়া হলোঃ

<h1 style="font-family:verdana;">This is a heading</h1>
<p style="font-family:courier;">This is a paragraph.</p>

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ

Text Size

CSS এর Font-Size বৈশিষ্ট্য এইচটিএমএল উপাদানটির জন্য টেক্সট এর আকার নির্ধারণ করে। অর্থাৎ Font Size কতো বড় বা ছোট হবে তা নির্ধারণ করে।

নিচে উদাহরণ দেয়া হলোঃ

<h1 style="font-size:300%;">This is a heading</h1>
<p style="font-size:160%;">This is a paragraph.</p>

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ

This is a heading


This is a paragraph.

Text Alignment

CSS এর Text-Align এই Property মূলত একটি টেক্সটের Alignment করতে ব্যবহৃত হয়। অর্থাৎ টেক্সট Align এর ৩ টি এর মধ্যে Center, Left, Right কোনটি হবে তা বোঝানোর জন্যে এটির ব্যবহার করা হয়।

নিচে উদাহরণ দেয়া হলোঃ

<h1 style="text-align:center;">Centered Heading</h1>
<p style="text-align:center;">Centered paragraph.</p>

Result ব্রাউজারে নিচের মতো প্রদর্শন করবেঃ

Centered paragraph.
